# SmartCart – Customer Segmentation using ML Clustering

## Overview
Unsupervised ML project to segment 2,240 retail customers 
into 4 distinct groups using KMeans and Agglomerative 
Clustering for targeted marketing strategies.

## Tech Stack
Python, Scikit-learn, Pandas, NumPy, Matplotlib, 
Seaborn, PCA, KMeans, KneeLocator

## Key Steps
- Feature Engineering (Age, Total Spending, Tenure)
- Outlier Removal & OneHot Encoding
- PCA for 3D Dimensionality Reduction
- Optimal K via Elbow Method + Silhouette Score (K=4)
- KMeans & Agglomerative Clustering
- Cluster Profiling & Business Insights

## Results
Identified 4 customer segments:
- High Income + High Spenders (Premium)
- Low Income + Low Spenders (Budget)
- High Income + Partner households
- Low Income + Solo households
